# Sign Language Bridge

AI-powered real-time communication system for Deaf and Hard-of-Hearing patients in hospitals.

## Features

- Real-time hand sign recognition
- Live webcam prediction
- Text-to-speech communication
- Medical phrase mapping
- Emergency visual alerts

## Tech Stack

- HTML
- CSS
- JavaScript
- Python
- FastAPI
- MediaPipe
- Scikit-learn

## Supported Signs

- A
- B
- HELP
- WATER
- PAIN

## Run Project

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
